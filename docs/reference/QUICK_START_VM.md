## ğŸ¯ Usage

### Via Wrapper Script
```bash
# Run with VM (fast, default)
./etamil program.qmz

# Run with LLVM (legacy, slower)
./etamil program.qmz --compile

# Show generated bytecode
./etamil program.qmz --bytecode

# Verbose output
./etamil program.qmz --verbose

# Use optimized build
./etamil program.qmz --release
```

### Via Cargo
```bash
# Default VM mode
cargo run --bin etamil_compiler -- program.qmz

# LLVM mode
cargo run --bin etamil_compiler -- program.qmz --llvm
```

### Via Direct Binary
```bash
# Build release binary
cd etamil_compiler
cargo build --release --bin etamil_compiler

# Run it
./target/release/etamil_compiler program.qmz
```

## â“ FAQ

### Q: Will my old programs still work?
**A**: Yes! LLVM mode still works. Add `--llvm` flag.

### Q: Is the VM fast enough?
**A**: Yes! 100x faster startup, good execution speed.

### Q: Do I need LLVM installed?
**A**: Not anymore! VM doesn't need external tools.

### Q: Can I distribute the binary?
**A**: Yes! Build with `cargo build --release`.

### Q: What if my program doesn't run?
**A**: Try `--llvm` mode. VM needs parser updates for some features.
