 ARCHITECTURE INTEGRATION
═══════════════════════════════════════════════════════════════════════════════

eTamil Language (.qmz)
    ↓
[Lexer: 103 tokens] → [Parser] → [Codegen: LLVM IR]
    ↓
File I/O Operations (CSV, JSON, text)
    ↓
CryptoHandler (Optional encryption)
    ↓
Database System (Relational & NoSQL)
    ↓
CommandExecutor (Query & compile ops)
    ↓
Output (Results, reports, LLVM IR)

   $ cd etamil_compiler

3. RUN eTAMIL EXAMPLES
